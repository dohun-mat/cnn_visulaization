# Guided Backpropagation을 이용하면, 예측에 중요한 역할을 한 픽셀들에 대한 정보를 추출할 수 있습니다. 이 기법은 역전파 과정에서 양의 그래디언트만 전파되도록 설계되어 있습니다. 따라서, 양의 그래디언트는 해당 픽셀이 해당 클래스의 예측에 기여한 정도를 나타내게 됩니다.
